# Myce Brand Guidelines

## Brand Essence

### Brand Personality
- **Authoritative** - Command respect in fintech
- **Calm** - No hype, no noise
- **Inevitable** - Feels like the obvious choice
- **Trustworthy** - Enterprise-ready, serious about money
- **Human** - Founder-to-founder, not corporate

### Brand Promise
Myce removes single-provider fragility from your revenue operations.

---

## Visual Identity

### Logo System

**Primary Logo**: Full wordmark with circle enclosure (2.png)
- Use on WHITE and LIGHT backgrounds (navigation, footer, light sections)
- Minimum width: 120px digital, 1 inch print
- Clear space: Equal to height of the "m"

**Secondary Logo**: Wordmark only (1.png)
- Use on PRIMARY BLUE backgrounds (dark sections, colored backgrounds)
- Also for space-limited contexts (favicons, social media)
- Minimum width: 80px digital

**Logo Usage Rules:**
- Never stretch or distort
- Never change colors
- Never add effects or shadows
- Never rotate
- Use 2.png on white/light backgrounds
- Use 1.png on blue/dark backgrounds

## Color System (60-30-10 Rule)

### Primary Colors (60% usage)

**Myce Blue** - Primary Brand Color
- HEX: `#0043FF`
- RGB: `0, 67, 255`
- Usage: Hero sections, primary CTAs, key messaging, trust elements

**Myce Blue 90**
- HEX: `#1A54FF`
- Usage: Hover states, active elements

**Myce Blue 10**
- HEX: `#E6EDFF`
- Usage: Subtle backgrounds, light sections

**Pure White**
- HEX: `#FFFFFF`
- Usage: Backgrounds, negative space, card surfaces

**Off White**
- HEX: `#F9FAFB`
- Usage: Subtle section separators, alternative backgrounds

**Gray 50**
- HEX: `#F9FAFB`
- Usage: Hover states on white

**Gray 100**
- HEX: `#F3F4F6`
- Usage: Borders, dividers

**Gray 600**

**Gray 900**
- HEX: `#111827`

## Typography

### Font Family

**Primary**: Inter
- Clean, neutral, professional
- Excellent readability at all sizes
- Modern without being trendy

**Weights Used:**
- Regular (400) - Body text
- Medium (500) - Emphasis, labels
- Semibold (600) - Subheadings
- Bold (700) - Headings, CTAs

### Type Scale

**Display (Hero Headlines)**
- Size: 56px / 64px / 72px (mobile/tablet/desktop)
- Weight: Bold (700)
- Line height: 1.1
- Letter spacing: -0.02em (tight)

**H1 (Main Headlines)**
- Size: 40px / 48px / 56px
- Weight: Bold (700)
- Line height: 1.2
- Letter spacing: -0.01em

**H2 (Section Headlines)**
- Size: 32px / 36px / 40px
- Weight: Bold (700)
- Line height: 1.3

**H3 (Subsection Headlines)**
- Size: 24px / 28px / 32px
- Weight: Semibold (600)
- Line height: 1.4

**Body Large**
- Size: 18px / 20px
- Weight: Regular (400)
- Line height: 1.6
- Max width: 65ch

**Body Regular**
- Size: 16px
- Weight: Regular (400)
- Line height: 1.6
- Max width: 70ch

**Body Small**
- Size: 14px
- Weight: Regular (400)
- Line height: 1.5

**Caption**
- Size: 12px
- Weight: Medium (500)
- Line height: 1.5
- Letter spacing: 0.01em
- Often uppercase for labels

**Spacing Scale:**
- 4px - xs (0.25rem)

- 16px - md (1rem)
- 24px - lg (1.5rem)

- 48px - 2xl (3rem)
- 64px - 3xl (4rem)
- 96px - 4xl (6rem)
- 128px - 5xl (8rem)

**Section Spacing:**
- Between sections: 80px mobile / 112px desktop

## Design Principles

### 1. Calm Authority
- Generous white space
- Clear hierarchy
- No unnecessary animation
- Professional without being sterile

### 2. Founder-Focused
- Direct, honest language
- No marketing speak
- Respect user intelligence
- Address real pain points

### 3. Enterprise Quality
- Precise alignment
- Consistent spacing
- High-quality imagery
- Attention to detail

### 4. Performance First
- Fast loading
- Optimized images
- Clean code
- Minimal dependencies

---

## Photography & Imagery

### Style
- **Avoid**: Stock photos of fake offices, forced smiles, staged computers
- **Use**: Abstract data visualizations, clean dashboard mockups, geometric patterns
- **Treatment**: Minimal, high contrast, professional

### Data Visualization
- Use brand colors purposefully
- Primary: Myce Blue for key metrics
- Accent: Electric Cyan for positive changes
- Neutral: Grays for secondary data

---

## Voice & Tone

### Writing Principles

**Do:**
- Short sentences
- Active voice
- Specific numbers
- Real problems
- Clear outcomes

**Don't:**
- Buzzwords (revolutionary, disruptive, next-gen)
- Hype (amazing, incredible, best)
- Vague claims (high-performance without context)
- Repeat "revenue orchestration" (once max)

### Messaging Hierarchy

**Primary**: Single-provider risk is dangerous
**Secondary**: Myce adds intelligence to payments
**Proof**: Metrics, results, real outcomes

## Digital Applications

### Website
- Clean, spacious layouts
- 60% white space and neutral backgrounds
- 30% Myce Blue for key sections and CTAs
- 10% Electric Cyan for interactive elements

### Iconography
- Stroke-based, not filled
- 2px stroke weight
- Rounded corners
- Simple, geometric

### Animations (when needed)
- Subtle, purposeful
- 200-300ms duration
- Ease-out timing
- Never distract from content
