# Myce Landing Page

A modern, high-performance landing page for Myce - a checkout and payments platform built for businesses that have outgrown single-provider risk.

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type safety and better developer experience
- **Tailwind CSS** - Utility-first CSS framework
- **Inter Font** - Clean, professional typography

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Project Structure

```
/app
  /components       # All React components
  page.tsx         # Main homepage
  layout.tsx       # Root layout
  globals.css      # Global styles
/public            # Static assets
```

## Components

- **Navigation** - Fixed header with CTA
- **Hero** - Main value proposition
- **TrustSignals** - Key metrics
- **ProblemSection** - Pain points
- **ShiftSection** - Introducing the solution
- **WhatMyceDoesSection** - Core capabilities
- **StripePayPalSection** - Positioning
- **HowItWorksSection** - Process flow
- **ResultsSection** - Impact metrics
- **WhoItsForSection** - Target audience
- **FinalCTA** - Conversion point
- **Footer** - Site footer
